Welcome to Mi(j)Chat

Instructions Below

Mi(j)Chat is a secure and anonymous CLI chat application that uses end-to-end encryption. Messages are only stored temporarily when the recipient is offline, and are deleted after delivery.

Connecting to the Server

To connect and start using Mi(j)Chat, run the application with the appropriate arguments:

	Register:
	./CliChatClient -r -u your_username -s server_address:port

	Login:
	./CliChatClient -l -u your_username -s server_address:port

Add --ignore-ssl if you're using a self-signed certificate.
Add --unicode if you want Unicode character support.

Navigation Keys

  • Backspace - Delete last character
  • Tab - Autocomplete previously entered names
  • Esc - Clear autocomplete memory
  • Up/Down - Browse message history
  • Page Up/Down - Scroll chat window

Main Commands

  • /add-group user1 user2 ... - Create a temporary end-to-end encrypted group chat
  • To send a message: simply type username message

Security Notes

  • RSA key pairs are generated on registration. Your private key is password-protected and saved locally.
  • Each chat session uses a securely exchanged AES key.
  • Server does not store any messages or user data after delivery.

For best experience, keep your private key secure and never share your password with anyone.
